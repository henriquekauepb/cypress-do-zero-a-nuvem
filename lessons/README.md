# Cypress-do-zero-a-nuvem


##### Documentação de projeto do Curso Cypress do zero a nuvem.

## Pré requisitos:

É necessário possuir o Node.js e o NPM instalados na máquina.
> Foi utilizada a versão `v24.12.0` do Node e `11.6.2` do NPM. Recomendo utilizar a mesma versão ou superiores.


## Instalação:
> Execute o comando `npm install cypress`.
